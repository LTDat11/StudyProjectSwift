# SwiftUI iOS App – Learning Project

This is a learning project built by following the YouTube video  
**“SwiftUI Course for Beginners – Create an iOS App from Scratch”**.

The main goal of this project is to understand the basics of **SwiftUI** and how to build a simple iOS application using **Xcode**.

---

## Project Goals

- Learn how to build UI using **SwiftUI**
- Understand the structure of an iOS project
- Practice running an app on the iOS Simulator
- Learn basic state management and navigation

---

## What I Learned

- Creating a SwiftUI project in Xcode
- Building UI with code (`Text`, `Image`, `Button`, `VStack`, `HStack`)
- Managing state with `@State`
- Screen navigation using `NavigationView` and `NavigationLink`
- Organizing code into simple views and models

---

## Technologies Used

- Swift
- SwiftUI
- Xcode

---

## How to Run

1. Open the project in **Xcode**
2. Select an iPhone Simulator
3. Press **Run (⌘R)** to build and run the app

---

## Notes

- This project is for **learning purposes only**
- The code focuses on understanding SwiftUI, not advanced architecture
- The project can be extended with MVVM, API integration, or data persistence
